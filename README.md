# 🎸 Nirvana Fan Website

This project is a website developed using HTML, CSS, and JavaScript, dedicated to the band Nirvana. The main goal was to practice web development concepts such as responsiveness, layout structuring, and user interaction.

---

## 🚀 Features

* 📱 Responsive navigation menu (mobile & desktop)
* 🎨 Modern layout using CSS and Flexbox
* 🎥 Multimedia section with video and subtitles
* 📊 Discography table
* 📝 Contact form with JavaScript validation
* 🔗 External links to songs

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM manipulation & events)
* Git & GitHub
